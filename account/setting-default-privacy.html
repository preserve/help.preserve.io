Setting a default privacy level will allows you to specify which bookmarks will show up by default when you go to Preserve (all your bookmarks, only your private bookmarks, or only your public bookmarks).

To change this setting, visit the settings page and choose a different Default Privacy setting. By default, it is set to All Bookmarks.

NOTE: Although setting a different default privacy level will change which bookmarks show up on your homepage, all bookmarks will still show up by default during a search.
