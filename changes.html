  • Feature (Apr 2) - Added a default privacy filter. In the settings, you can now specify which privacy level you'd like to see by default (All Bookmarks, Private Bookmarks, Public Bookmarks).
  • Little "Fix" (Mar 18) - When you search for a Bookmark or view a tag, it the query or tag will now be shown in the page title.
  • Enhancement (Mar 17) - Added the ability to recache your bookmarks! From the bookmark edit page, click re-cache. This will delete your old cached copy and then create a new up-to-date one. Only use this feature if you know that the page is still available.
  • Enhancement (Mar 5) - Added RSS to User Profiles. You can append .rss to the user profile. For example preserve.io/u/zackfern.rss.
  • Feature (Jan 26) - Added the ability to view a text and JSON version of your latest bookmarks. Visit bookmarks.txt or bookmarks.json to check it out. These files can be saved for backup purposes.
