We currently offer an official Preserve extension for Google Chrome, as well as a bookmarklet which will work in all web browsers.

The Boookmarklet

Preserve this Page ‚Üê Drag this to your bookmarks bar

Using the bookmarklet is simple, just follow these steps:

  1. When you're at a website you'd like to bookmark, simply click the bookmarklet from your toolbar.
  2. A window will popup which will contain the title of the page and allow you to tag the bookmark.
  3. Once you've tagged and set any cache / privacy settings on the bookmark, click Create Bookmark.
  4. Once you're forwarded to a page which says "Bookmark successfully created!" you can close the window.
